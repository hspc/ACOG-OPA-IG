This guide references the following value sets.

Value Set Name Descripton url
FPAR Pregnancy Status Pregnancy status at time of encounter. ValueSet-pregnancyStatus
FPAR Sexual Intercourse History Values used to indicate weather or not a patient has ever had sexual intercourse. ValueSet-virginNonVirgin
FPAR Contraceptive Type This values contains both clinical and non-clinical contraceptive types. ValueSet-contraceptiveChoice
FPAR Contraception Not Used Reason Reasons why contraception is not being used. ValueSet-reasonNoContraceptiveChosen
FPAR Contraceptive Provision Method Indicating how contraception provided at a high-level during a clinical encounter. ValueSet-contraceptionProvisionMethod
FPAR Pregnancy Intent Values describing patient intentions to become pregnant. ValueSet-pregnancyIntention
FPAR Pregnancy Status Reporting Method Method used to report and/or determine pregnancy status. ValueSet-pregnancyStatusReportingMethod
FPAR Clinical Provider Role Clinical provider occupational roles. ValueSet-fparProviderRole
FPAR Annual Household Income Ranges Household annual income range. ValueSet-fparAnnualHouseholdIncomeRanges
FPAR Insurance Coverage Type Source of Payment Typology codes for a low granularity description of Insurance coverage. ValueSet-coverageType
FPAR English Proficiency Patient's level of proficiency in English in four domains: listening, writing, reading, or speaking. ValueSet-englishProficiency
Smoking Status Classification of a patient's smoking behavior. This value set is consistent with the Smoking Status codes used by ONC for Meaningful Use ValueSet-fparSmokingStatus
Payer Typology Used to describe the type of the primary payer for the visit. ValueSet-primaryInsuranceClassification
EthnicGroup The codes for the ethnicity categories - 'Hispanic or Latino' and 'Non Hispanic or Latino' - as defined by the OMB Standards for Maintaining, Collecting, and Presenting Federal Data on Race and Ethnicity, Statistical Policy Directive No. 15, as revised, October 30, 1997. https://www.hl7.org/fhir/us/core/ValueSet-omb-ethnicity-category.html
BirthSex Codes for assigning sex at birth as specified by the Office of the National Coordinator for Health IT (ONC) https://www.hl7.org/fhir/us/core/ValueSet-us-core-birthsex.html
Race The codes for the concepts 'Unknown' and 'Asked but no answer' and the the codes for the five race categories - 'American Indian' or 'Alaska Native', 'Asian', 'Black or African American', 'Native Hawaiian or Other Pacific Islander', and 'White' - as defined by the OMB Standards for Maintaining, Collecting, and Presenting Federal Data on Race and Ethnicity, Statistical Policy Directive No. 15, as revised, October 30, 1997 . https://www.hl7.org/fhir/us/core/ValueSet-omb-race-category.html
FPAR RNA Detection Ordinal values used to result RNA detection based laboratory tests. For example LOINC code 5018-7, HIV 1 RNA [Presence] in Unspecified specimen by Probe and target amplification method. ValueSet-fparHIV1RnaDetectedNotDetected
FPAR Positive Negative Ordinal values for Positive and Negative results. ValueSet-negativePositive
FPAR Organism Detection Ordinal lab results that reflect microorganism presence. ValueSet-organismDetectionOrdLabResult
FPAR Detected, Not Detected, Equivocal Inconclusive Ordinal values for Detected and Not Detected results, with the inclusion of Equivocal and Inconclusive. ValueSet-fparDetectedNonDetectedEqInd
FPAR Detected, Not Detected Ordinal values for Detected and Not Detected results. ValueSet-fparDetectedNotDetected
FPAR Antibody Reactivity Positive Negative Ordinal values used to result antibody based laboratory tests. For example LOINC code 5221-7, HIV 1 Ab [Presence] in Serum by Immunoblot (IB). ValueSet-fparHIVAbReactiveNonReactive
FPAR Pap Smear Tests Laboratory tests for PAP Smear tests, used to retrieve and populate dates for PAP Smear labs. ValueSet-papSmearTests
FPAR HIV Tests Laboratory tests for HIV, used to retrieve and populate dates for Human Immunodeficiency Virus labs. ValueSet-hivTests
FPAR Human Papilloma Virus Tests Laboratory tests for Human Papilloma virus, used to retrieve and populate dates for Human Papilloma virus labs. ValueSet-humanPapillomaVirusTests
FPAR Chlamydia Trachomatis Tests Laboratory tests for Human Papilloma virus, used to retrieve and populate dates for Human Papilloma virus labs. ValueSet-chlamydiaTrachomatisTests
FPAR Neisseria Gonorrhoeae Tests Laboratory test exclusively for Neisseria gonorrhoeae, used to retrieve and populate dates for Neisseria gonorrhoeae labs. ValueSet-neisseriaGonorrhoeaeTests
FPAR Chlamydia trachomatis and Neisseria Gonorrhoeae combined Tests Laboratory tests which test for both Chlamydia trachomatis and Neisseria gonorrhoeae, used to retrieve and populate dates for Chlamydia trachomatis and Neisseria gonorrhoeae labs. ValueSet-chlamydiaTrachomatisAndNeisseriaGonorrhoeaeCombinedTests
FPAR Pregnancy Tests Contains the LOINC codes for the labs used for Preganacy status determination. ValueSet-pregnancyTests
